# ollama addon for brewbot
Talk with your bot by using ai
# use guide (new version)
<p>1 get ollama: https://ollama.com/</p>
<p>2 get ai model you want</p>
<p>3 edit config.js (use notepad)</p>
<p>4 set ai model you got</p>
<p>5 set players bot can talk to</p>
<p>6 edit prompt if you want</p>
<p>7 put addon in brewbot/addons</p>
<p>8  /msg 'yourbot' (text) to talk with ai</p>
